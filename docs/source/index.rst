Welcome to the Hy2DL Documentation
==================================

This is the documentation for the **Hy²DL** Python package. The source code is available on `GitHub <https://github.com/eduardoAcunaEspinoza/Hy2DL>`_.

In this documentation, you will find:

 - A :doc:`usage guide <usage/getting_started>` to help you get started with the library.
 - A :doc:`guide <usage/models>` to the different models implemented in the library.  
 - A :doc:`configuration reference <usage/config>` with all available experiment arguments.  
 - The complete :doc:`API reference <api/hy2dl>` for modules, classes, and functions.
